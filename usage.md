### How to use /ftp

Make a post request with the following body replacing the sample data with your own.

    {
    	"config": {
    
    		"host": "host IP address",
    		"type": "FTP",
    		"port": "21",
    		"username": "username",
    		"password": "userpassword"
    	},
    
    	"paths": ["source/path", "destination/path"]
    }
